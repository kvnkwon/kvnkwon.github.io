Little bit about Javascript

Why so popular?

Did you know that every web browser supports javascript? Yes- even the dreaded Internet Explorer. This is one of the reasons why javascript is so popular. Many websites utilize javascript for cross browser functionality.

Javascript can be a server side language. With node.js, developers are able to create websites completely in javascript without the use of other languages such as PHP and Ruby! No longer is javascript limited to front-end development!

Object oriented design! Sure there are other languages that incorporate OOP, but it's just another trait that makes javascript awesome. You can easily create objects in javascript and incorporate OOP design patterns.

Want to pick up javascript? No problem! Another reason for its popularity is because of the low difficulty in learning it! Javascript's syntax is close to English making it very readable.

Weaknesses?

There were some complaints about security issues with javascript, which could signal a red flag for some. Javascript snippets on websites run on client servers immediately which can result in the user's system being exploited.
